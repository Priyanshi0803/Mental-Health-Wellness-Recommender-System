# Mood-Based Wellness Recommender

A personalized wellness recommender system suggesting **meditations, podcasts, readings, and music** based on your current mood. Built with **Streamlit** and CSV-based catalogs for easy customization.

- Multi-type content recommendations:

  - **Readings** for mental health, inspiration, or positivity  
  - **Music tracks** for mood boosting, calming, or reflection  
- CSV-based content catalogs for **easy expansion**  
- Quick, **user-friendly Streamlit interface**
